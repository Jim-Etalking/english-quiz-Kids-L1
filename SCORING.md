# 評分規則

這份規則用於 ROOM + PARK 兩關，目標是判斷孩子目前比較接近：

- Pre-Starter
- Pre-Starter Plus
- Starter

如果結果達到 Starter，才建議進入第三關，用第三關再判斷 Starter / L1。

## 評分原則

這個測驗是給小小孩使用，所以評分不採用「錯一次就大扣分」的方式。孩子可能因為手指誤觸、觀察畫面、操作拖曳不熟而點錯，因此評分會保留容錯。

每題分數由三個因素組成：

```text
單題得分 = 題目難度權重 x 錯誤次數係數 x 反應時間係數
```

最後再換算成百分比：

```text
總分百分比 = 實際得分 / 最高可得分 x 100
```

## 題目難度權重

| 難度 | 權重 | 題型說明 |
|---|---:|---|
| 簡單 | 1 | 基礎物件辨識，例如 sun、door、slide、tree |
| 中等 | 2 | 顏色、較不直覺物件或干擾較多的辨識，例如 blue bear、red ball、seesaw、sandbox、bird、dog |
| 較難 | 3 | 拖曳與方位指令，例如 put...on、put...in、in front of |

## 錯誤次數係數

| 錯誤次數 | 係數 | 說明 |
|---:|---:|---|
| 0 | 1.00 | 穩定理解 |
| 1 | 0.90 | 容許小朋友誤觸或試探 |
| 2 | 0.75 | 可能需要多一點提示或練習 |
| 3 | 0.55 | 理解不穩定 |
| 4 次以上 | 0.40 | 題目概念可能尚未掌握 |

## 反應時間係數

Tap 題：

| 秒數 | 係數 |
|---:|---:|
| 0-4 秒 | 1.00 |
| 4-8 秒 | 0.90 |
| 8-12 秒 | 0.75 |
| 12 秒以上 | 0.60 |

Drag 題：

| 秒數 | 係數 |
|---:|---:|
| 0-6 秒 | 1.00 |
| 6-12 秒 | 0.90 |
| 12-18 秒 | 0.75 |
| 18 秒以上 | 0.60 |

Drag 題給較長時間，因為它同時包含聽懂指令、找到物件、拖曳操作與放到正確位置。

## 等級判斷

ROOM + PARK 完成後，系統會計算：

- Overall Score：全部題目的加權總分
- Vocabulary：非拖曳題的詞彙與物件辨識
- Response Speed：反應時間穩定度
- Following Directions：拖曳與方位指令題

推薦等級使用以下規則：

| 條件 | 建議等級 | 後續 |
|---|---|---|
| Overall < 60 | Pre-Starter | 先加強基礎單字與簡短聽力 |
| Overall 60-74 | Pre-Starter Plus | 接近 Starter，但建議補強穩定度 |
| Overall >= 75 且 Following Directions >= 65 | Starter | 建議進入第三關，評斷 Starter / L1 |

## 為什麼要加 Following Directions 門檻

如果孩子只會找圖，但不太能完成 put / on / in / in front of 這類指令，代表他可能還沒穩定達到 Starter。

因此即使總分達到 75，只要方位指令表現低於 65，仍先不直接判為 Starter。

## 星等換算

報表中的星星依百分比換算：

| 百分比 | 星等 |
|---:|---:|
| 90-100 | 5 星 |
| 75-89 | 4 星 |
| 60-74 | 3 星 |
| 45-59 | 2 星 |
| 0-44 | 1 星 |

## 目前題目定位

ROOM + PARK 是第一段評測，用來分辨 Pre-Starter / Starter。

建議流程：

```text
ROOM + PARK
  -> Pre-Starter / Pre-Starter Plus / Starter
  -> 如果是 Starter，進入第三關
  -> 第三關再判斷 Starter / L1
```

這樣可以避免孩子還沒準備好就被推進更高難度，也能讓程度較好的孩子有機會往 L1 評估。
